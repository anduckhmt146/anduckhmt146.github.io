Here is solutions to explains idea for others.

# 1. How to answer the question ?

1. Context

- We are solving the problem about + \_\_\_
- With the main goals is + \_\_\_

2. Contrains

- The main contrains is +\_\_\_

3. Options

- Naive solution: Pros, Cons, Improvement.
- Optimization
- Test

4. Recommendation

- Based on the constraints and trade-offs, I recommend **_ because _**.

5. Action

- Initial variables\_\_\_.
- Iterate through\_\_\_.
- Update \_\_\_.
- Return \_\_\_.

6. Result

- Achievements.

# 2. Time-based approach

1.  **30-seconds:**

- Context → Best Option → Result

- Goals: Can you summarize clearly?

2.  **2-mins:**

- Context → Constraints → Options (Naive → Optimized → Test) → Recommendation → Action

- Goals: Can you reason and choose tradeoffs?

3.  5-7 mins:

- Context → Constraints → Options (Naive → Optimized → Test) → Recommendation → Action

- Goals: Can you design, justify, and scale?

# 3. Linking Word

- Because of...

- As a result...

- Firstly, Secondly, Finally,...

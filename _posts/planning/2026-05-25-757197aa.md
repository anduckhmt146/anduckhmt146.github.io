How to be focus and work efficiency to reduce stress.

# 1. Start with a Clear Target

- What is the exact outcome?
- What does “done” look like?
- What is the most important task today?

# 2. Reduce Context Switching

Bad:
- Coding → Slack → Email → Meeting → Coding

Better:
- Batch similar tasks together
- Check messages only at specific times

Example:
- Email at 11 AM and 4 PM only

# 3. Use a “Next Action” System

When stuck, define the smallest next step.

Instead of: “Build project”

Use:
- Open document
- Write outline
- Draft intro
- Create first function

Momentum creates motivation.

# 4. How to prevent doing boring task ?

- See the positive aspect when doing something.

# 1. Framework translate stakeholder

- Hiểu rất sâu + collect data đầy đủ 1 cái trước khi communicate.

- Communicate vừa đủ surface cho nhiều stakeholders: thứ họ cần thôi + scope.

# 2. Framework for writing docs

- Hiểu từng section là gì của 1 cái docs, và cần đọc những gì.

- Giao tiếp: Context + Problem + Acceptance Criteria, khoan go to Solutions.

# 3. Framework for Active Listening

- Remove distractions, tập trung vô speaker.

- Clear a point của người nói.

- Thắc mắc gì cứ hỏi.

# 4. Framework for Understand Other Member Scope
